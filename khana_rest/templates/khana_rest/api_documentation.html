API DOCUMENTATION

  • /khana_rest/food_items/ to see all food items available
  • /khana_rest/food_items/1 to see the first food item
  • /khana_rest/orders/ to see all orders available
  • /khana_rest/orders/1 to see the first order
  • /khana_rest/shops/ to see all shops available
  • /khana_rest/shops/1 to see the first shop
  • /khana_rest/dates/ to see all dates
  • /khana_rest/dates/1 to see the first date
  • /khana_rest/users/ to see all users
  • /khana_rest/users/1 to see the user
  • /khana_rest/temp_users/ to see all temporary users waiting for approval
  • /khana_rest/temp_users/1 to see the first temporary user
